# ECOMOCAP
This repository is dedicated to the ECOMOCAP project about markerless motion capture for clinical, ergonomics and sports application. It covers the calibration and synchronization of a 4-camera system, pose and shape estimation and biomechanical analysis. Codes are provided in Python.
